# Skill Pack S11 — Sales & BD: SPEC

**ID:** S11 | **Tier:** 2 | **Status:** Scaffold in v1.0; harden in v1.x

## Purpose

Brain-fed pipeline that writes itself. Outreach skill patterns. Lumina widget as the front door for inbound. Sales output is first-class output in the partner output ledger — no demoting commercial work to second-class status.

## Scope

Pipeline construction from Brain signals (customer mentions, contract events, conversation threads). Outreach prompt templates. Lumina widget integration for inbound lead capture and qualification. Closed-deal Brain page convention. Long-tail entitlement attribution for sales work (the YOLO post explicitly addresses the "salespeople deserve attribution too" question).

## Allowed tools

Curator MCP (Brain reads + writes). Lumina AI. Email / messaging (Cloud track) or 4thtech dMail (Privacy track).
